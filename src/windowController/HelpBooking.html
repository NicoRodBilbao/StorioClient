Booking Management: Help

General Description

This window allows you to perform basic maintenance operations of the bookings data. The user can use the controls available in the window to create, search, modify and delete bookings in the application. the application. The user can also generate a report containing a list of the bookings with their corresponding data.

Operations

Create Bookings

To create a booking you must, first of all, click once on the Create button. Then, you must enter the description, start date and end date in their respective fields. You must also select the packs you want to book. Once you have completed the above actions, you must click the Create button again. If no error occurs, the reservation will be created and will appear in the reservation table shown in the window.

Modify Bookings Data

To modify a booking you have to click once on the Modify button. You can click in the table row that you want to modify now or before clicking Modify button first time. Then, you must change at least one of the fields. Once you have completed the above actions, you must click the Modify button again. If no error occurs, the booking will be modified and will appear in the reservation table shown in the window.

Delete Booking

To delete a booking you have to click once on the Delete button. You can click in the table row that you want to delete now or before clicking Delete button first time. You can also put the id of the booking that you want to delete without selecting in the table. Then you click the Delete button again. If no error occurs, the booking will be deleted and will disappear in the booking table shown in the window.

Print booking list

Clicking on the Help button in the menu above and then clicking on Print Report will open a window where you can see a document containing a list of the bookings that exist in the application. Using the buttons at the top of the window you can print or save the report.

Exit the application

Pressing the button Go back in the menu above the application will close.
